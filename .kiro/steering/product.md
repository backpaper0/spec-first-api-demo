# Product Overview

cc-sdd（Claude Code Spec-Driven Development）とClaude CodeのPlanモードを組み合わせた仕様駆動開発のデモプロジェクト。AI-DLC（AI Development Life Cycle）の実践として、KiroスタイルのSpec-Driven Developmentをどのようにアプリケーション開発へ適用するかを示す。

## Core Capabilities

- **仕様駆動のAPI開発**: 要件 → 設計 → タスク → 実装の3フェーズ承認ワークフローでRESTful APIを開発
- **品質ゲートの自動化**: Spotless・Checkstyle・SpotBugs・JaCoCoによる静的解析とカバレッジチェックをビルドに組み込む
- **AIとのコラボレーション**: Claude Codeのspec/steeringによるプロジェクト知識管理とPlanモードによる設計レビュー

## Target Use Cases

- AI支援によるSpec-First開発プロセスのデモンストレーション
- cc-sddワークフロー（仕様初期化→要件→設計→タスク→実装）の学習・参照実装

## Value Proposition

仕様をコードより先に書き、AIとの対話で品質を担保することで、設計の手戻りと技術的負債を防ぐ開発スタイルを体現したリポジトリ。
